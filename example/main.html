GLEIT

super easy interactive html animations.

with GLEIT you can easily do animations based on page scroll, mouse movement, etc. 

<script src="https://unpkg.com/gleit"></script>
<p data-gleit='{
  "0vh": {"scale": 1},
  "20vh": {"scale": 2}}'>hellow</p>

<script>
  window.addEventListener('load', function() {
    gleit.animate().on(gleit.verticalScroll());
  });
</script>
